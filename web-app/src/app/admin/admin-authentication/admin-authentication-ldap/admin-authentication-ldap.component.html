URL An LDAP URL describes an LDAP search operation that is used to retrieve information from an LDAP directory and begins with the protocol prefix "ldap".
Authentication
Bind DN The bind DN is basically the credential you are using to authenticate against an LDAP. When using a bind DN it usually comes with a password associated with it, which is set in the bind credentials. Bind Credentials Password used to authenticate against an LDAP.
Search
Search Base The search base defines the starting point for the search in an LDAP. Search Filter This requires knowledge of the LDAP search filter grammar.
