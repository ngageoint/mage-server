Open ID Connect Credentials
Client Identifier This is obtained from the OAuth 2.0 server (typically once you register your app). The client identifier is unique to the OAuth 2.0 server, and is not considered secret. Client Secret This is obtained from the OAuth 2.0 server (typically once you register your app), and should be kept secret. Issuer Entity that issues a set of Claims.
Endpoints
Authentication Endpoint Endpoint to obtain authorization grant from the resource owner. Token Endpoint Endpoint to exchange an authorization grant for an access token. User Profile Endpoint Endpoint where the user profile is obtained.
