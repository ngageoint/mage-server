Credentials
Client Identifier This is obtained from the OpenID server (typically once you register your app). The client identifier is unique to the OpenID server, and is not considered secret. Client Secret This is obtained from the OpenID server (typically once you register your app), and should be kept secret.
URLs
Authorization URL Endpoint to obtain authorization grant from the resource owner. Token URL Endpoint to exchange an authorization grant for an access token. User Profile URL Once an access token is obtained, this endpoint will be accessed to obtain user information. This may also be referred to as API server. Issuer URL Endpoint of your identity provider. This is the entity that issues a set of claims.
Access
Scope One or more scope values indicating which parts of the user's account you wish to access (comma-separated). This must include oidc, and will be enforced by MAGE if omitted. Advanced
User Profile
Profile ID Property The id property that is returned in the user profile. Defaults to 'sub'. Display Name Property The display name property that is returned in the user profile. Defaults to 'name'. Email Name Property The email property that is returned in the user profile. Defaults to 'email'.
