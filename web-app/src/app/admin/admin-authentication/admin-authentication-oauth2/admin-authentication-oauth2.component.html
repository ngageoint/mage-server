OAuth 2.0 Credentials
Client Identifier This is obtained from the OAuth 2.0 server (typically once you register your app). The client identifier is unique to the OAuth 2.0 server, and is not considered secret. Client Secret This is obtained from the OAuth 2.0 server (typically once you register your app), and should be kept secret.
Endpoints
Authorization Endpoint Endpoint to obtain authorization grant from the resource owner. Token Endpoint Endpoint to exchange an authorization grant for an access token. User Info Endpoint Endpoint where the user profile is obtained. Advanced
These settings are intended for users with extensive knowledge of OAuth 2.
