Account Lock
Enabled {{accountLockChoice.title}} MAGE can lock user accounts after a number of invalid login attempts for a certain amount of time.
Account Lockout Threshold Number of invalid login attempts, after which the users account will be locked. Account Lockout Interval (Seconds) Amount of time in seconds a users account will remain locked after reaching number of invalid login attempts. Enable Max Lockout {{maxLockChoice.title}} MAGE can disable user accounts if a user locks their account a certain number of times.
>Max Lockout Threshold Number of times a user account can be locked before it is disabled. Disabled accounts must be re-enabled by an admin. If left blank this feature is disabled.
