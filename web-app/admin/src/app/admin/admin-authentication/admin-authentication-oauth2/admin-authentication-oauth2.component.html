Credentials
Client Identifier This is obtained from the OAuth 2.0 server (typically once you register your app). The client identifier is unique to the OAuth 2.0 server, and is not considered secret. Client Secret This is obtained from the OAuth 2.0 server (typically once you register your app), and should be kept secret.
URLs
Authorization URL Endpoint to obtain authorization grant from the resource owner. Token URL Endpoint to exchange an authorization grant for an access token. User Profile URL Once an access token is obtained, this endpoint will be accessed to obtain user information. This may also be referred to as API server.
Access
Scope Zero or more scope values indicating which parts of the user's account you wish to access (comma-separated).

PKCE (Proof Key for Code Exchange)
Helps prevent CSRF and authorization code injection attacks.
